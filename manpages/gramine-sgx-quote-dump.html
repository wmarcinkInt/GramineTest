gramine-sgx-quote-dump – Display SGX quote structure

Synopsis

gramine-sgx-quote-dump [OPTION]… FILE

Description

gramine-sgx-quote-dump displays internal structure of an SGX quote contained in FILE.

Command line arguments

-h, --help

Display usage.

-m, --msb

Print hex strings in big-endian order.
